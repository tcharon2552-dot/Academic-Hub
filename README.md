# Academic Hub

Academic Hub is a vertical AI subscription service concept for researchers and labs.

The product uses New API as the backend model gateway, but the user-facing product is positioned around research workflows:

- Paper reading
- Literature review
- Academic writing
- Reviewer response drafting
- Research code and data analysis
- Lab collaboration
